# Transport Order Management API

API REST para la gestión de órdenes de transporte de una empresa de movilidad.

## Características

- Gestión completa de órdenes de transporte
- Gestión de conductores
- Asignación de conductores a órdenes con archivos adjuntos
- Autenticación JWT
- Documentación Swagger/OpenAPI
- Pruebas unitarias e integración
- Contenedor Docker

## Tecnologías

- Java 17
- Spring Boot 3.1.5
- PostgreSQL 15
- JWT para autenticación
- MapStruct para mapeo
- Flyway para migraciones
- Docker & Docker Compose

## Requisitos Previos

- Docker y Docker Compose
- Java 17 (para desarrollo)
- Maven 3.8+
